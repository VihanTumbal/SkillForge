# 🚀 SkillForge

**AI-Powered Developer Skill Tracking Platform**

A modern full-stack application that helps developers track their skills, set learning goals, and get AI-powered career insights.

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, TypeScript, TailwindCSS
- **Backend**: Node.js, Express.js, TypeScript
- **Database**: MongoDB
- **AI**: Google Gemini API
- **Auth**: JWT

## 🚀 Quick Start

### Prerequisites

- Node.js 20+
- MongoDB
- Google AI API Key

### Setup

1. **Clone and install:**

```bash
git clone <repo-url>
cd SkillForge
npm install
```

2. **Backend setup:**

```bash
cd backend
cp .env.example .env
# Add your MongoDB URI and Google AI API key
npm run dev
```

3. **Frontend setup:**

```bash
cd frontend
npm run dev
```

4. **Access:**

- Frontend: http://localhost:3000
- Backend: http://localhost:5001
